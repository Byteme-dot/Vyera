# Vyera

Vyera is a Personal Knowledge Management System (PKMS) built using Python. It is designed as a local desktop application that allows users to capture, organize, and retrieve information efficiently.

The application provides a structured environment where users can create notes, organize them into folders, and quickly search through their knowledge base. Vyera stores all data locally using SQLite, ensuring fast performance and full control over user data.

The graphical interface is built with CustomTkinter to provide a modern and clean desktop experience.

## Features

- Create, edit, and delete notes
- Organize notes into folders
- Search notes by keywords
- Local storage using SQLite
- Clean and modern GUI using CustomTkinter

## Planned Features

- Markdown support
- Tagging system
- Graph view of connected notes
- File attachments
- Export notes to PDF
- Plugin system

## Tech Stack

- Python
- CustomTkinter
- SQLite
- Git & GitHub
